## 交互流程

- 浏览目录 → 选择文档分类 → 查看文档内容
- 搜索 → 显示匹配结果列表 → 点击查看
- 文档内容支持 Markdown 渲染
- 支持代码高亮和复制

## 权限说明

- 访问权限：所有用户（包括未登录）

## 后端 API 依赖

- `GET /api/docs` - 获取文档列表
- `GET /api/docs/:id` - 获取文档内容
- `GET /api/docs/search` - 搜索文档
