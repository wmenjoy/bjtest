## 如何创建新的决策记录

1. 复制模板: `cp _template.md YYYY-MM-DD-主题-类型.md`
2. 填写内容（背景、决策、备选方案、理由等）
3. 提交PR进行评审
4. 更新本index.md索引

**命名规范**:
- 格式: `YYYY-MM-DD-主题-[architecture|feature].md`
- 示例: `2025-11-26-workflow-redesign-feature.md`
