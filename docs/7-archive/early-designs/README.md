## 📝 归档规范

### 归档条件

将设计文档归档到 `early-designs/` 需满足以下条件之一：

1. **设计已实现**: 功能已完成开发，设计文档仅供历史参考
2. **设计已调整**: 实际实施时采用了不同的设计方案
3. **设计已废弃**: 功能不再计划实现

### 归档时必须

- ✅ 保留原文件名（便于追溯）
- ✅ 添加归档说明到本README
- ✅ 记录归档原因和时间
- ✅ 如有替代文档，提供链接
