# 高保真原型

本目录包含产品的高保真设计稿和交互原型。

## 高保真原型用途

高保真原型用于：
- 展示最终视觉效果
- 定义品牌视觉元素
- 演示交互细节
- 用户测试和反馈收集

## 设计工具

推荐使用以下工具：
- **Figma** - 在线协作设计（推荐）
- **Sketch** - macOS设计工具
- **Adobe XD** - Adobe设计工具
- **Axure** - 交互原型工具

## 文档格式

### 方式1: 链接到在线设计工具

```markdown
# UI高保真设计

## Figma设计稿

**链接**: [Figma - 测试平台UI设计](https://figma.com/file/...)

## 包含页面

1. 登录页面
2. 仪表盘
3. 测试用例管理
...

## 版本历史

- v1.0 (2025-11-26): 初始版本
```

### 方式2: 存储设计资源

如需在仓库中存储设计文件：
- PNG/JPG截图 → `docs/assets/designs/screenshots/`
- PDF导出 → `docs/assets/designs/pdf/`
- 设计源文件（不推荐） → 使用云存储，在文档中添加链接
