# 界面线框图

本目录包含产品各页面的线框图设计。

## 线框图用途

线框图用于：
- 展示页面布局和结构
- 确定元素位置和层级
- 规划信息架构
- 快速迭代设计方案

## 文档模板

创建新的线框图文档时，请使用以下模板：

```markdown
# [页面名称] 线框图

**版本**: 1.0
**最后更新**: YYYY-MM-DD
**设计者**: 姓名

---

## 页面功能

简述页面的核心功能和用户目标。

## 页面结构

[ASCII艺术线框图或Figma链接]

## 关键元素

1. **元素名称** - 功能说明
2. **元素名称** - 功能说明

## 交互说明

- 点击行为
- 状态变化
- 页面跳转

## 相关流程

- 链接到user-flows/中的相关流程文档

---

**设计者**: 姓名
**评审状态**: 待评审/已评审/已实现
```
