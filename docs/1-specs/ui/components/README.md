# UI组件设计规范

本目录包含所有可复用UI组件的技术实现规范。

## 目录说明

每个组件规范文档应包含：

1. **技术规范**
   - React组件接口（Props, State）
   - TypeScript类型定义
   - 依赖项

2. **样式规范**
   - CSS类名约定
   - 主题变量使用
   - 响应式样式

3. **交互行为**
   - 事件处理逻辑
   - 状态变化规则
   - 可访问性（a11y）

4. **数据流**
   - Props数据传递
   - 事件回调
   - 状态管理

5. **示例代码**
   - 基础用法
   - 高级用法
   - 最佳实践
