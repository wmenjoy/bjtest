# 布局设计规范

本目录包含页面布局系统和响应式设计规范。

## 目录说明

布局规范文档应包含：

1. **网格系统**
   - 列数定义（12列/24列）
   - 间距规范
   - 对齐规则

2. **响应式断点**
   - Mobile: < 768px
   - Tablet: 768px - 1024px
   - Desktop: > 1024px

3. **容器规范**
   - 最大宽度
   - 内边距
   - 外边距

4. **间距系统**
   - Spacing tokens
   - Margin/Padding scale
